# PISCAR FORA

> **Magia Arcana de Nível 4**  
> **Custo de Recordação:** 1

Faça uma Jogada de Conjuração de Magia (12). Em caso de sucesso, gaste uma Esperança para se teletransportar para outro ponto que você possa ver dentro do alcance Longo. Se houver criaturas dispostas dentro do alcance Muito Próximo, gaste uma Esperança adicional para cada criatura para levá-las com você.
